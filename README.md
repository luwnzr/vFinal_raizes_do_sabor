## 📖 Sobre o Projeto

O **Raízes do Sabor** é um website responsivo desenvolvido para um restaurante fictício. O projeto combina um design elegante com funcionalidades de filtragem de cardápio.

Este projeto foi desenvolvido em **equipe** durante o **Curso de Desenvolvimento de Software**, com o objetivo de aplicar conhecimentos práticos de Front-end e uso de frameworks.

## 🚀 Funcionalidades

### 🏠 Início
- Banner principal com identidade visual marcante e navegação intuitiva.

### 📜 Sobre Nós
- História fictícia do restaurante e sua proposta de culinária afetiva.

### 🍽️ Cardápio Interativo (Filtros Dinâmicos)
Utilização de JavaScript para filtrar os pratos sem recarregar a página.
**Categorias:**
* Entradas, Petiscos, Pratos Principais, Saladas, Sobremesas e Bebidas.

### 📍 Contato & Localização
- Endereço e mapa integrado.
- Horário de funcionamento.
- Área para feedback de clientes.
- Botão de retorno ao início.

## 🛠️ Tecnologias Utilizadas

* **HTML5 & CSS3**
* **JavaScript:** Lógica de manipulação do DOM para os filtros.
* **[Bootstrap 5.3.2](https://getbootstrap.com/):** Framework para layout responsivo (Grid System) e componentes.
* **Bibliotecas de Ícones:** Bootstrap Icons e Font Awesome.
* **Tipografia:** Google Fonts (*Playfair Display*).
